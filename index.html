HL1.css

CSS library based on Half-Life 1 UI.

# Installation

You can install HL1.css simply by pasting this to your html `head` tag: 

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/TAERABYTE/HL1.css@main/css/HL1.min.css">

# Dialog

Options

The Half-Life 1 manual text details the game's controls, including primary fire (Mouse 1), secondary fire (Mouse 2), and weapon switching (Mouse Wheel, 'Q'). It also explains movement keys like WASD or arrow keys, jump (Spacebar), and actions such as crouching (Control) and using the flashlight (F). The manual covers the game's main menu options and recommends the Hazard Course for new players.
